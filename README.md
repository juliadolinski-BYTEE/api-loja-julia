# API Loja

Projeto Full-Stack desenvolvido com Node.js, Express e front-end em HTML/CSS/JavaScript.

## 📌 Funcionalidades

- CRUD completo de produtos
- Cadastro de categorias
- Interface web simples
- Persistência em arquivo JSON

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- HTML
- CSS
- JavaScript (Fetch API)

## ▶️ Como executar o projeto

1. Instale as dependências:
npm install

2. Execute o servidor:
node index.js

3. Acesse no navegador:
http://localhost:3000

## 🔗 Endpoints da API

### Produtos
- GET /produtos
- POST /produtos
- PUT /produtos/:id
- DELETE /produtos/:id

### Categorias
- GET /categorias
- POST /categorias
